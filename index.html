🔐 1. 로그인 및 접속

관리자 페이지에 접속하여 부여받은 ID와 PW를 입력해 주시기 바랍니다. 기업 보안 강화를 위해 로그인은 2단계 인증을 거치게 됩니다.

2단계 인증 가이드

  1. 아이디와 비밀번호를 입력한 후 로그인 버튼을 클릭합니다.
  2. 화면에 표시된 등록 이메일 주소를 확인하고 인증번호 요청 버튼을 클릭합니다.
  3. 담당자 이메일로 발송된 인증번호 6자리를 확인합니다.
  4. 인증번호를 입력창에 넣고 인증 확인 버튼을 클릭하여 인증을 완료하면 메인 화면으로 이동합니다.
💡 Tip 'IP 인증' 방식을 설정해 두면, 최초 인증 후 3개월간 해당 IP에서는 추가 인증 없이 편리하게 로그인할 수 있습니다.

📝 4. 신규 검사 등록

화면 우측 상단의 채용검사 등록 버튼을 클릭하여 새로운 검사 프로젝트를 생성할 수 있습니다.

Step 1. 검사 관리명 & 2차 인증 설정
  • 채용검사명기업 내부에서 관리하실 채용검사명을 작성해 주시기 바랍니다. (최대 50자 제한)
  • 기간 입력 실제 검사가 진행되는 일시(분 단위 설정 가능)를 선택합니다. ※ 검사 종료일은 검사 생성일로부터 최대 3년까지만 설정할 수 있습니다.
  • 검사 상품지원자가 응시하고자 하는 상품을 선택합니다. (상품에 따른 기본 응시 가능 횟수와 검사 시간이 자동 설정되며, 가급적 기본값 유지를 권장합니다.)
🛡️ 2차 인증 설정 안내

더욱 안전하고 대리 응시를 방지하는 신뢰도 높은 검사 진행을 위해 본인 확인용 2차 인증 기능이 기본 적용됩니다.

  • 인증 수단: '휴대폰 번호', '이메일', '휴대폰 번호/이메일(택 1)' 중 선택할 수 있으며, 선택하신 옵션에 따라 향후 응시자 명단 등록 시 필수 입력 정보가 달라집니다.
  • 온라인 검사: 기본값 사용으로 자동 설정됩니다.
  • 오프라인 검사: 기본값 미사용으로 처리되며, 2차 인증 설정이 불가(비활성화)합니다.
  • 응시 가능 횟수 지원자의 인터넷(모바일) 등 환경 불안정으로 검사 응시가 비정상적으로 중단되었을 때, 다시 접속할 수 있도록 허용하는 횟수를 의미합니다. 단, 검사를 완전히 종료하고 최종 제출을 완료한 이후에는 다시 응시할 수 없습니다.
  • 검사 타이틀 문구최대 30자까지 작성할 수 있습니다. 작성하신 타이틀 문구 뒤에 '검사를 시작합니다.'라는 시스템 문구가 자동 표기됩니다.
  • 유의사항응시자가 검사 시작 전 가장 먼저 확인하게 되는 페이지입니다. 텍스트를 자유롭게 수정할 수 있으며(이미지 첨부 불가), 서류 마감일이나 향후 면접 일정 등 추가 공지사항으로 활용하시기 좋습니다.
  • 문의처 노출담당자 정보에 기재된 연락처가 안내 문자 최하단에 함께 노출됩니다. (단, 실제 문자 발신 번호는 02-6377-2503으로 고정되어 발송됩니다.)
🚨 응시자 등록 후 설정 변경 제한

등록된 데이터의 검증 및 개인정보 보안을 강화하기 위해 응시자가 1명이라도 시스템에 등록된 이후에는 일부 설정 변경이 엄격히 제한됩니다.

  • 2차 인증 사용 여부: [사용] ➔ [미사용]으로의 변경은 가능합니다. 단, [미사용] ➔ [사용]으로는 변경할 수 없습니다.
  • 2차 인증 수단 변경: 최초에 선택한 인증 수단(예: 휴대폰)을 도중에 다른 수단(예: 이메일)으로 변경할 수 없습니다.
  • Step 1 기본 정보 제한: Step 3(응시자 등록) 완료 후에는 검사 시간, 검사 기간(시작일), 검사 언어, 검사 구분(온/오프라인)의 수정이 원천 차단됩니다.

Step 2. 데이터 관리 (개인정보 파기 정책)

데이터 보관 기간을 명확하게 설정하고 관리하실 수 있도록 데이터 관리 단계가 신설되었습니다. 기업에서 설정하신 '개인정보 보관 기간'이 경과하면 해당 데이터는 시스템에 의해 영구 파기되거나 비식별화 처리됩니다.

개인정보 보관 기간 자동 설정 안내

기업의 편의와 개인정보 보호 규정 준수를 위해, 개인정보 보관 기간이 시스템 기본값으로 자동 세팅됩니다.

  • 기본 보관 기간 (자동 설정): 초기 검사 생성 시 이름/결과지는 3년, 연락처는 180일로 자동 지정됩니다.
  • 설정 가능 기간 (사전 제공 옵션):
    • 연락처(휴대폰, 이메일): 60일, 90일, 120일, 150일, 180일 중 선택
    • 이름/결과지: 90일, 180일, 1년, 2년, 3년 중 선택
⚠️ 보관 및 파기 관련 중요 주의사항
  • 응시자가 1명이라도 등록된 이후에는 설정된 '보관 기간'을 절대 수정할 수 없습니다.
  • 필수 정책에 따라, 연락처 보관 기간은 이름 및 결과지 보관 기간을 초과하여 길게 설정할 수 없습니다.
  • 설정된 보관 기간이 경과하여 완전히 파기된 데이터는 어떠한 경우에도 복구할 수 없습니다. (따라서 별도 보관이 필요하신 경우 반드시 기간 만료 전 직접 백업해 주시기 바랍니다.)
※ 참고: 사람인 대행 서비스를 이용하시는 기업의 경우 시스템 기본값이 제공되지 않으므로, 검사 생성 시 관리자가 직접 보관 기간을 지정하셔야 합니다.

Step 3. 응시자 등록

응시자 등록 단계에서는 개별 등록, 응시자 엑셀 등록, 면접번호 등록 모드를 진행할 수 있습니다. 설정하신 검사 방식(온라인/오프라인)에 따라 최적화된 화면이 제공됩니다.

일괄 등록 방법 (Excel)

대량 데이터의 정확도를 보장하기 위해 '전체 성공 또는 전체 실패(All-or-Nothing)' 무결성 방식이 적용됩니다. 엑셀 파일 내 단 한 건이라도 필수값이 누락되거나 형식이 올바르지 않으면 전체 응시자 등록이 실패 처리됩니다.

  1. 왼쪽 엑셀 양식 다운로드 버튼을 클릭하여 최신 엑셀 템플릿 파일을 받습니다.
  2. 양식에 맞추어 응시자 정보를 입력합니다. (이때, 선택하신 2차 인증 수단에 해당하는 정보가 필수로 기입되어야 합니다.)
  3. 응시자 엑셀 등록 버튼을 클릭하여 작성하신 파일을 시스템에 업로드합니다.
📋 엑셀 등록 시 필수 유의사항
  • 반드시 시스템에서 제공하는 업데이트된 양식을 다운로드하여, 예시와 동일한 데이터 형식으로 작성해 주시기 바랍니다. (형식 불일치 시 등록이 불가합니다.)
  • 필수 항목: 수험번호, 이름
  • 2차 인증 필수값: 이메일, 휴대폰 번호는 기본적으로 선택 항목이나, 해당 수단으로 2차 인증을 사용할 경우 반드시 그에 맞는 연락처 정보를 포함해야 합니다.
  • 특수문자 원천 차단: 개인정보 오기입 방지 및 시스템 안정성을 위해 수험번호, 지원분야, 비고란에 @ 와 . 기호 입력이 원천 차단됩니다. 수험번호는 가급적 개인정보가 노출되지 않는 고유 번호로 설정해 주시기 바랍니다.
  • 안정적인 업로드를 위해 1회 대량 등록 시 500명 이내로 분할하여 등록하시기를 권장합니다.

개별 등록 방법 (직접 입력)

  1. 응시자 행 추가+ 버튼을 클릭하여 응시자 정보를 한 명씩 화면에 입력합니다.
  2. 모든 입력이 끝난 후 저장 후 다음 버튼을 클릭하여 명단을 확정합니다.
🚨 응시자 목록 전체 노출 제한

개인정보 보호 정책에 따라, 응시자 명단을 저장한 후 해당 페이지에 재진입하실 경우 기존 리스트는 화면에 노출되지 않으며 우측 상단에 '현재 등록 인원: 00명'으로만 간략히 요약 표시됩니다. 등록된 응시자의 개별 정보 수정 및 삭제는 상단 응시자 관리 탭 메뉴를 통해서만 안전하게 진행하실 수 있습니다.

🔒 응시자 접속 화면 및 2차 인증 유의사항

응시자가 제공받은 링크로 검사 사이트에 접속할 때, 사전에 기업 관리자가 설정한 2차 인증 수단으로 엄격한 본인 확인 절차가 진행됩니다.

  1. 재접속 시 재인증: 응시자가 검사 도중 응시를 일시 중단하고 다시 접속하는 경우, 보안을 유지하기 위해 선택된 인증 수단으로 2차 인증을 재진행해야 합니다.
  2. 인증 정보 변경 불가: 응시자 본인이 직접 2차 인증 정보(휴대폰 번호, 이메일 등)를 접속 화면에서 수정할 수 없습니다. 반드시 기업 담당자가 사전에 등록 및 설정한 정보로만 인증이 매칭되어 가능하므로, 최초 등록 전 정확한 정보 입력이 필수입니다.
⚠️ [중요] 2차 인증 5회 오류 시 접근 제한

응시자가 본인 확인 2차 인증 번호 입력을 5회 이상 연속으로 실패할 경우, 해당 시점으로부터 30분간 검사 사이트 접근이 강제 차단됩니다.
※ 이 차단 조치는 관리자 권한으로도 임의 해제(초기화)가 불가능하므로, 응시자에게 사전에 정확한 본인 정보 입력을 각별히 안내해 주시기 바랍니다.

📝 4. 신규 검사 등록

Step 4. 안내 문자/메일 발송

안내 문자 및 메일 발송 단계에서는 앞서 명단을 등록한 응시자들에게 검사 참여를 독려하는 문자메시지나 이메일을 직접 발송할 수 있습니다.

  • ✅ 개인정보 수집 동의 안내: 노출된 내용을 꼼꼼히 확인하고 동의를 체크합니다. (동의하지 않으실 경우 시스템을 통한 발송이 제한됩니다.)
  • ✅ 이메일/문자메시지 선택: 발송하실 안내 수단을 선택합니다.
  • ✅ 발송 유형 선택:
    • 즉시: 지금 바로 전체 대상자에게 발송합니다.
    • 예약: 원하시는 전송 날짜와 시간을 지정하여 예약 발송합니다.
  • ✅ 메일/메시지 발송 내역 확인: 미리보기 클릭 시 발송 내역 미리보기 및 문자메시지 발송 내용 수정이 가능하며, 발송내역 보기 클릭 시 실제 발송 일시를 확인할 수 있습니다.
  • ✅ 발송/예약 실행: [즉시] 또는 [예약] 체크박스 클릭 후 아래 안내 발송 및 발송 예약(전체응시자 대상) 버튼을 클릭하면 안내가 발송(혹은 예약) 됩니다.
⚠️ 예약 발송 주의사항

문자 발송 예약은 현재 설정 후에 예약 일시 수정 및 취소가 불가능하며, 예약된 자동 발송이 이루어지기 전에는 해당 팝업에서 즉시 발송 기능을 추가로 이용하실 수 없습니다. 예약의 변경이나 삭제 등을 희망하실 경우 시스템 문의처로 개별 문의하시기 바랍니다.

문자/이메일 내용 수정 방법

  • 수정 화면 진입화면 하단에 있는 이메일/문자메시지 영역의 미리보기 버튼을 클릭합니다.

📝 4. 신규 검사 등록

Step 5. 응시 현황 수신 설정

담당 관리자가 전체 검사의 진행 상황(미응시 인원, 응시 완료 인원 등)을 매일 오전 9시와 검사 마감 직후에 메일이나 문자로 요약해 정기적으로 받아볼 수 있는 똑똑한 편의 기능입니다.

📊 5. 결과 확인 및 응시자 관리

Step 1. 홈 화면

채용검사 관리(홈 화면)에서는 전체적인 현황을 한눈에 파악하고, 성적표 간편 다운로드 및 미응시자 대상 안내 재발송 업무를 신속하게 수행할 수 있습니다.

1) 응시 현황 확인 및 안내 재발송

📊 응시 현황 확인 및 안내 재발송

홈 화면에 위치한 검사 리스트 우측의 전체인원 텍스트를 클릭하면 해당 검사의 '응시자 리스트'가 팝업으로 노출되며 구체적인 현황 파악이 즉시 가능합니다.

  • 현황 확인전체인원 숫자를 클릭하여 노출된 팝업(최대 100명 노출)에서 지원자 상태별(대기 중, 응시 중, 검사 종료) 인원 규모를 직관적으로 파악할 수 있습니다.
  • 안내 재발송응시자에게 문자/메일 재발송이 필요한 경우, 좌측 체크박스로 대상 인원만 특정하여 선택한 후 재발송 처리를 진행할 수 있습니다. (주로 미응시자의 검사 독려 시 매우 유용하게 사용됩니다.)
✏️ 응시자 정보 수정 및 삭제

응시자 리스트 팝업 내에서 개별 응시자의 정보를 빠르게 수정하거나 불필요한 응시자 데이터를 삭제할 수 있습니다.

🚨 주의

응시자 등록 이후에는 기본적으로 수험번호와 성명 수정이 엄격히 제한됩니다. '응시 중' 또는 '응시 완료' 상태로 넘어간 응시자는 휴대전화 번호 및 이메일 또한 수정이 불가하도록 데이터 잠금 처리됩니다.
마찬가지로 검사를 시작하여 응시 중이거나 응시 완료(검사 종료)된 대상자는 시스템 데이터의 정합성을 위해 임의 삭제가 절대 불가합니다.

2) 결과표(PDF/Excel) 다운로드

홈 화면 우측 '종합 리포트' 항목의 다운로드 버튼을 클릭하시면 PDF 다운로드, Excel 다운로드, 해석 가이드 다운로드 메뉴 팝업이 노출됩니다. 이곳에서 원하시는 유형의 파일을 클릭하여 결과를 쉽게 내려받을 수 있습니다.

* 대량 다운로드 유의사항: 검사 완료자가 50명을 초과하는 대규모 채용 전형의 경우, 홈 화면에서의 간편 PDF 다운로드 기능이 비활성화 처리됩니다. 이 경우에는 상단 메인 메뉴의 응시자 관리 탭으로 이동하시어 상세 다운로드를 진행해 주시기 바랍니다.

📊 5. 결과 확인 및 응시자 관리

Step 2. 응시자 관리 화면

접근 방법: 홈 화면 현황 내 응시인원 숫자를 직접 클릭하시면 해당 검사의 상세 제어가 가능한 응시자 관리 탭으로 즉시 이동합니다.

* 홈 화면 최상단의 응시자 관리 메인 탭을 직접 누르실 경우, 진행된 모든 검사에 속한 기업의 전체 응시자 목록이 통합되어 노출됩니다.

1) 부가 기능(상세 필터) 및 안내 발송

💡 부가 기능 상세 설명
  • 대상 필터링응시 시작일, 응시 여부(완료/미응시 등), 이름 조건 등 세부적인 맞춤형 검색 기능을 강력하게 지원합니다.
  • 결과별 필터링'인성', '부적응', '직무능력', '신뢰도' 등 특정 점수 컷라인이나 성향을 기반으로 한 세밀한 심층 결과 필터링을 제공합니다.
  • 발송 관리원하는 조건으로 정밀하게 필터링된 대상자들만 골라내어 단체 안내 문자 및 메일 발송 처리를 빠르고 정확하게 수행할 수 있습니다.

2) 상세 결과표(PDF/Excel) 다운로드

응시자 관리 탭에서 결과표 다운로드를 진행하실 때는 시스템의 안정적이고 빠른 처리를 보장하기 위해 PDF와 Excel 파일 모두 최대 50명 단위로 분할(끊어서) 다운로드가 처리됩니다.

🚨 주의!

화면 상의 개인결과표 메일발송 및 개인결과표 문자발송 기능 버튼은 검사 참여를 독려하는 일반적인 '응시 안내' 버튼이 절대 아닙니다. '최종 응시 결과(심층 성적표)'를 해당 지원자 본인에게 직접 발송해버리는 기능이오니, 민감한 평가 결과가 지원자에게 오발송되지 않도록 가급적 사용을 자제하여 주시기 바랍니다.

3) 결과지 내 블라인드 처리

콘솔 메뉴의 결과지 설정 버튼을 클릭하시면 최종 결과지 문서에서 제외할 항목을 직접 고를 수 있는 옵션 팝업이 노출됩니다. 해당 팝업에서 체크하여 선택하신 개인 정보 항목은 최종 다운로드 결과지 문서 상에서 삭제(블라인드) 표기되어 안전하게 인쇄됩니다.

* 응시자 정보에 수험번호와 면접번호를 둘 다 입력하신 경우, 결과지에는 면접번호가 무조건 우선적으로 표기됩니다. (면접번호 대신 수험번호를 표기하고 싶으신 경우에는 이 블라인드 기능을 활용하여 면접번호를 블라인드 처리해 주시면 수험번호가 정상 표기됩니다.)

4) 부적격 판별 기준 커스텀 설정

콘솔 메뉴의 판별 기준 설정 버튼을 누르면 판별 기준 제어 팝업이 뜹니다. 이 기능을 영리하게 활용하면 각 기업의 고유한 내부 채용 기준에 맞게 부적격 점수 컷라인 기준을 유연하고 섬세하게 조정하실 수 있습니다.

  • 시스템 기본값: 종합점수/인성/직무 40점 미만, 신뢰도 80점 미만, 부적응성 1개 이상 항목 80점 이상 시 자동으로 부적격 처리됩니다.
  • 조정 예시: 우리 회사의 인성검사 판별 기준을 60점으로 상향 조정하여 세팅할 경우, 60점 미만을 득점한 지원자는 모두 '부적격' 판정으로 분류됩니다.

* 주의사항: 점수 기준 설정은 가급적 지원자가 응시를 시작하기 전에 조정을 완료하셔야만 최종 결과표 도출 시 논리적인 오류가 발생하지 않습니다. 응시가 이미 진행된 이후 부득이하게 커트라인 수정이 필요하신 경우에는 시스템 관리자 문의처(02-6377-2306)로 즉각 연락을 부탁드립니다.

5) 개별 인원 결과표(상세) 브라우저 보기

응시자 리스트 가장 우측에 위치한 보기 버튼을 클릭하시면, 무거운 파일 다운로드 과정 없이 빠르고 가벼운 브라우저 팝업 상에서 해당 응시자의 개별 상세 검사 결과를 즉시 확인하실 수 있습니다.

❓ 6. FAQ

FIT 관리자 시스템을 실무에서 사용 중이신 담당자분들께서 가장 자주 문의하시는 주요 사항과 명확한 답변을 모아두었습니다. 시스템 이용 중 문제가 발생하였을 시 가장 우선적으로 확인해 주시기 바랍니다.

Q. 관리자 계정 아이디와 비밀번호를 분실했습니다. 어떻게 찾아야 하나요?
아이디 및 비밀번호 분실 시 기업의 정보 보안상의 이유로 시스템 화면 상에서 직접 찾기/초기화 기능은 별도로 제공되지 않습니다. 사람인 시스템 문의처(02-6377-2306, 2308) 또는 담당자 이메일(consulting_test@saramin.co.kr)로 연락을 취해 주시면, 철저한 본인 확인 절차를 거친 후 안전하게 초기화를 지원해 드립니다.
Q. 응시자가 2차 인증을 계속 실패하여 사이트 접속이 차단되었다고 연락이 왔습니다. 관리자가 즉시 해제해 줄 수 있나요?
아니요, 시스템 보안 정책상 임의 해제가 불가능합니다. 강력한 대리 응시 방지 정책에 따라 2차 인증 번호 입력을 5회 이상 연속 실패하실 경우 자동으로 30분간 검사 사이트 접근이 강제 차단됩니다. 이는 기업 관리자 권한으로도 초기화가 불가하도록 시스템이 설계되어 있으므로, 30분이 완전히 경과한 뒤 등록된 정확한 본인 정보로 인증을 차분히 재시도하시도록 응시자에게 직접 안내해 주시기 바랍니다.
Q. 응시자 관리 목록에서 지원자들의 이름과 연락처가 *** 표시로 가려져서 전혀 보이지 않습니다.
민감한 개인정보 유출 사고를 방지하기 위해 관리자가 시스템에 접속 시 기본적으로 '보안 모드(데이터 마스킹)'가 자동으로 작동하기 때문입니다. 마스킹을 일시 해제하고 지원자의 전체 정보를 명확하게 확인하시려면 화면 우측 상단의 계정 메뉴 탭을 열어 상세 보기 모드 옵션으로 전환해 주시기 바랍니다.
Q. 엑셀 일괄 등록 시 자꾸 업로드에 실패합니다. 데이터 오류가 없는 인원만이라도 먼저 부분 등록할 수는 없나요?
현재 시스템은 대규모 데이터의 누락 및 꼬임 현상을 원천 방지하기 위해 '전체 성공 또는 전체 실패(All-or-Nothing)' 데이터 무결성 방식을 엄격하게 적용하고 있습니다. 엑셀 파일 내 단 한 명의 정보라도 필수값이 누락되거나 시스템 형식이 어긋날 경우 전체 명단 등록이 일괄 실패 처리됩니다. 다소 번거로우시더라도 오류 내역을 확인하시어 엑셀 데이터를 완벽하게 보완하신 뒤 한 번에 업로드해 주시기 바랍니다.
Q. 응시자 연락처 등 개인 정보를 잘못 입력했습니다. 지금 수정이 가능한가요?
해당 응시자가 아직 검사 접속 및 응시를 시작하지 않은 온전한 '대기' 상태라면 상단의 응시자 관리 메뉴에서 개별 정보 변경이 즉시 가능합니다. 단, 응시자가 이미 검사 화면을 열어 진행 중이거나 완료한 상태, 혹은 파기 기한이 이미 도래하여 일부 데이터가 시스템에서 지워진 상태에서는 정보 수정 기능이 엄격히 제한됩니다.
※ 주의: 신규 검사 생성 시 설정하신 '2차 인증 수단'과 '데이터 보관 기간'은 대상 응시자가 1명이라도 시스템에 등록되면 그 즉시 환경 설정 수정 버튼이 원천 차단됩니다.
Q. 개인정보 보관 기간(파기 기한)을 기업 사정에 맞춰 뒤늦게 연장할 수 있나요?
불가능합니다. 응시자가 1명이라도 시스템에 등록된 이후에는 처음에 설정된 데이터 보관 기간을 임의로 늘리거나 줄이는 조작을 할 수 없습니다. 설정된 기간 만료 시 데이터가 영구히 파기 복구 불가 처리되므로, 지원자 이력 보관이 필요하신 기업은 반드시 기한 만료 전에 미리 엑셀/PDF 형태로 로컬 PC에 백업을 진행해 주셔야 합니다.
Q. 성적 결과표(PDF) 다운로드 버튼을 눌러도 화면 반응이 없고 파일이 다운로드되지 않습니다.
이용 중이신 브라우저(크롬, 엣지 등)의 팝업 차단 기능이 켜져 있어 다운로드 창을 가로막고 있는지 확인해 주시기 바랍니다. 주소창 우측의 경고 아이콘을 눌러 해당 사이트의 팝업을 '항상 허용'으로 변경하신 뒤 다시 시도해 보시기 바랍니다. (시스템 과부하 방지를 위해 PDF 상세 다운로드는 한 번에 최대 50명까지만 체크하여 진행하실 수 있습니다.)
Q. 부적격 판별 기준 점수 컷라인은 언제 설정하거나 수정해야 안전한가요?
결과표 점수 도출 및 부적격 판별 로직의 오류를 완전히 방지하기 위해 반드시 지원자가 검사에 응시하기 전에 기준 세팅을 모두 완료해 주셔야 합니다. 검사가 이미 진행된 이후 부득이하게 커트라인 수정이 필요하신 경우에는, 번거로우시더라도 시스템 관리자 문의처로 즉각적인 기술 지원을 요청해 주시기 바랍니다.
Q. 회사 내부에서 자체적으로 시스템 테스트를 진행할 때 유의해야 할 사항이 있나요?
실제 채용 지원자와 테스트 데이터가 섞여 통계에 혼선을 주지 않도록, 내부 테스트용 검사 생성 시 검사명 혹은 응시자(테스터) 이름에 'TEST' 또는 '테스트'라는 단어를 명확하게 포함하여 기재해 주시기를 강력히 권장해 드립니다.
Q. 특정 검사를 완전히 삭제해버리거나, 응시 진행 도중 멈춘 인원을 강제로 채점/초기화 하려면 어떻게 하나요?
응시 횟수 초과 해제와 같은 기본적인 인원 관리 권한은 페이지 내비게이션에서 버튼 클릭만으로 쉽게 제어하실 수 있습니다. 그러나 데이터 보관 기간이 경과하여 파기가 이미 진행된 응시자에 대해서는 데이터 정합성을 위해 시험 초기화 기능이 완전 차단됩니다. 이 외의 전체 검사 그룹의 통째 삭제나 도중 하차 인원에 대한 강제 채점 등의 고급 권한 기능은 시스템 문의처(02-6377-2306)로 유선 요청해 주시면 상황 파악 후 신속하게 조치해 드리겠습니다.
Q. 대금 청구와 세금계산서 발행 절차가 어떻게 이루어지는지 궁금합니다.
기업의 월별 시스템 총 이용 내역을 안전하게 집계하여, 다음 달 첫 영업일에 상세 정산 내역서와 함께 담당자 메일로 안내해 드리고 있습니다. 수신하신 내역에 이상이 없음을 확인 회신해 주시면 세금계산서를 즉시 발행해 드리며, 이후 안내된 지정 계좌로 이용 대금을 입금해 주시면 모든 절차가 완료됩니다. (월 결제가 아닌 특정 건별 개별 정산이 필요하신 경우에는 별도 문의를 부탁드립니다.)
Q. 검사 비용에 대한 과금 청구 기준은 정확히 무엇인가요?
채용 검사가 최종 종료되어 정상적으로 결과지가 시스템에 도출(생성)된 응시자의 수에 한하여 비용이 청구(과금)됩니다. 명단에 등록만 해두고 응시하지 않은 인원(대기 중)이나, 검사 진행 도중 중도 포기하여 최종 결과지가 나오지 않은 불완전 인원에 대해서는 검사 비용이 전혀 발생하지 않으니 안심하셔도 됩니다.
