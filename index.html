사람인 F.I.T검사 관리자 AI매뉴얼

효율적인 인재 채용을 위한 FIT 검사 관리자 시스템 사용법을 안내드립니다.

0. 검사 관리자 등록

사람인 FIT 검사 이용 신청 시, 담당자 이메일로 관리자 페이지 URL, 계정 ID, 초기 비밀번호가 발송됩니다.

1. 서비스 이용 동의

  • 최초 접속 시 '서비스 이용 동의' 및 '개인정보 위탁 동의'가 필요합니다. 동의 후 탈퇴 전까지 해당 계약은 유효한 것으로 간주합니다.
최초 1회에 한하여 신규 계정 등록 비용 300,000원(VAT 별도)이 청구됩니다.

1. 로그인 및 접속

관리자 페이지에 접속하여 부여받은 ID와 PW를 입력해 주세요. 보안 강화를 위해 로그인은 2단계 인증을 거칩니다.

접속 정보

  • 관리자 페이지 URLhttps://v2.testfit.co.kr/company
  • 시스템 문의02-6377-2306, 2308

2단계 인증 가이드

  1. 아이디와 비밀번호를 입력한 후 [로그인] 버튼을 클릭합니다.
  2. 화면에 표시된 등록 이메일 주소를 확인하고 [인증번호 요청] 버튼을 클릭합니다.
  3. 담당자 이메일로 발송된 인증번호 6자리를 확인합니다.
  4. 인증번호를 입력창에 넣고 인증을 완료하면 메인 화면으로 이동합니다.
💡 Tip 'IP 인증' 방식을 설정해두면, 최초 인증 후 3개월간 해당 IP에서는 추가 인증 없이 편리하게 로그인할 수 있습니다.

2. 채용검사 관리 (메인 화면)

로그인 직후 보이는 메인(홈) 화면입니다.
현재 진행 중이거나 종료된 모든 검사와 현황을 한눈에 확인할 수 있습니다.

주요 기능 설명

항목 설명
검사 리스트 등록된 검사명, 기간, 상품명, ON/OFF 상태, 접속 URL 등 상세 정보를 확인합니다.
검색 기능 채용검사명, 상품명을 검색하여 등록된 채용검사를 확인할 수 있습니다.
검사 URL 확인 검사 페이지로 이동할 수 있는 URL을 확인할 수 있습니다. (클릭 시 자동 복사)
응시현황 확인 전체인원과 응시인원(검사 종료인원)을 실시간으로 확인할 수 있습니다.
종합 리포트 해당 검사의 전체 결과를 엑셀 파일 형태로 다운로드합니다.
응시자 추가 진행 중인(등록된) 검사에 새로운 응시자를 추가로 등록할 수 있습니다.

4. 신규 검사 등록

우측 상단의 [채용검사 등록] 버튼을 클릭하여 새로운 검사 프로젝트를 생성합니다.

Step 1. 채용 검사 관리명 설정
  • 채용검사명원하는 채용검사명을 작성해주세요. (최대 50자 제한)
  • 기간입력 검사 진행 일시(분 단위 설정 가능)를 선택합니다. ※ 검사 종료일은 생성일로부터 최대 3년까지만 설정할 수 있습니다.
  • 검사 상품응시하고자 하는 상품을 선택합니다. (기본 응시가능횟수와 검사시간이 자동 설정되며, 가급적 기본값 유지를 권장합니다.)
🛡️ 2차 인증 설정 안내

더욱 안전하고 신뢰도 높은 검사 진행을 위해 본인 확인용 2차 인증 기능이 적용됩니다.

  • 인증 수단: 휴대폰 번호, 이메일, 휴대폰 번호/이메일(택 1) 중 선택할 수 있으며 선택한 옵션에 따라 응시자 등록 시 필수 입력 정보가 달라집니다.
  • 온라인 검사: 기본값 ‘사용’으로 자동 설정됩니다.
  • 오프라인 검사: 기본값 ‘미사용’ (2차인증 설정 불가 / 비활성화)
  • 응시 가능 횟수 인터넷(모바일) 환경 불안정 등으로 검사 응시가 중단되었을 때 다시 접속할 수 있는 횟수를 의미합니다. 검사 제출 완료 이후에 다시 응시하는 것은 불가합니다.
  • 검사타이틀 문구최대 30자까지 작성 가능합니다. 작성한 타이틀 뒤에 "검사를 시작합니다."라는 문구가 자동 표기됩니다.
  • 유의사항응시자가 검사 시작 전 확인하는 페이지입니다. 자유롭게 수정 가능하며(이미지 불가), 서류 마감일, 면접 일정 등 추가 공지사항으로 활용 가능합니다.
  • 문의처 노출담당자 정보에 기재된 번호는 안내 문자 최하단에 노출됩니다. (단, 문자 발신 번호는 02-6377-2503으로 고정됩니다.)
🚨 응시자 등록 후 설정 변경 제한 등록된 데이터 검증 및 개인정보 보안 강화를 위해 응시자가 1명이라도 등록된 이후에는 일부 설정 변경이 엄격히 제한됩니다.
  • 2차 인증 사용 여부: [사용] → [미사용] 변경은 가능합니다. 단, [미사용] → [사용]으로는 변경할 수 없습니다.
  • 2차 인증 수단 변경: 최초 선택한 인증 수단(예: 휴대폰)을 다른 수단(예: 이메일)으로 변경할 수 없습니다.
  • Step 1 기본 정보 제한: Step 3(응시자 등록) 완료 후에는 검사 시간, 검사 기간(시작일), 검사 언어, 검사 구분(온/오프라인)의 수정이 원천 차단됩니다.
※ 설정을 부득이하게 변경하려면 등록된 전체 응시자를 삭제하거나 신규 검사를 다시 생성해야 합니다.

4. 신규 검사 등록

Step 2. 데이터 관리 (개인정보 파기 정책)

데이터 보관 기간을 명확하게 설정할 수 있도록 데이터 관리 단계가 신설되었습니다. 기업에서 설정한 '개인정보 보관 기간'이 경과하면, 해당 데이터는 영구 파기되거나 비식별화 처리됩니다.

💡 변경된 검사 생성 프로세스 Step 1. 검사명 ➔ Step 2. 데이터 관리(신설) ➔ Step 3. 응시자 등록 ➔ Step 4/5. 안내 및 현황

개인정보 보관 기간 자동 설정 안내

기업의 편의와 개인정보 보호 규정 준수를 위해, 개인정보 보관 기간이 시스템 기본값으로 자동 설정됩니다.

  • 기본 보관 기간 (자동 설정): 초기 검사 생성 시 이름/결과지는 3년, 연락처는 180일로 자동 지정됩니다.
  • 설정 가능 기간 (사전 제공 옵션):
    • 연락처(휴대폰, 이메일): 60일, 90일, 120일, 150일, 180일
    • 이름, 결과지: 90일, 180일, 1년, 2년, 3년
⚠️ 보관 및 파기 관련 중요 주의사항
  • 응시자가 1명이라도 등록된 이후에는 설정된 '보관 기간'을 절대 수정할 수 없습니다.
  • 필수 정책에 따라, 연락처 보관 기간은 이름 및 결과지 보관 기간을 초과하여 설정할 수 없습니다.
  • 설정된 보관 기간이 경과하여 파기된 데이터는 어떠한 경우에도 복구할 수 없습니다. (별도 보관이 필요한 경우 만료 전 직접 백업 요망)
※ 참고: 대행 서비스를 이용하는 기업의 경우 기본값이 제공되지 않으므로, 검사 생성 시 직접 보관 기간을 지정해야 합니다.

4. 신규 검사 등록

Step 3. 응시자 등록

응시자 등록 단계에서는 응시자 개별등록, 엑셀 파일 일괄 등록, 면접번호 등록을 진행하실 수 있습니다. 검사 방식(온라인/오프라인)에 따라 화면이 구분됩니다.

일괄 등록 방법 (Excel)

데이터의 정확도를 위해 '전체 성공 또는 전체 실패(All-or-Nothing)' 방식이 적용됩니다. 엑셀 파일 내 단 한 건이라도 필수값이 누락되거나 형식이 올바르지 않으면 전체 응시자 등록이 실패 처리됩니다.

  1. 왼쪽 [양식 다운로드] 버튼을 클릭하여 업데이트된 엑셀 파일을 받습니다.
  2. 양식에 맞춰 응시자 정보를 입력합니다. (선택한 2차 인증 수단에 해당하는 정보 필수)
  3. [응시자 엑셀 등록]을 클릭하여 작성한 파일을 업로드합니다.
📋 엑셀 등록 시 필수 유의사항
  • 업데이트된 양식을 다운로드하여 예시와 동일한 형식으로 작성해 주세요. (형식 불일치 시 등록 불가)
  • 필수 항목: 수험번호, 이름
  • 2차 인증 필수값: 이메일, 휴대폰 번호는 기본적으로 선택 항목이나, 문자/이메일 2차 인증을 사용할 경우 선택하신 옵션에 맞는 정보가 반드시 포함되어야 합니다.
  • 특수문자 원천 차단: 개인정보 오기입 방지를 위해 수험번호, 지원분야, 비고란에 @ 와 . 입력이 원천 차단됩니다. 수험번호는 가급적 개인정보가 포함되지 않도록 주의해 주세요.
  • 1회 대량 등록은 500명 이내를 권장합니다.

개별 등록 방법 (직접 입력)

  1. [응시자 행 추가+] 버튼을 클릭하여 한 명씩 정보를 입력합니다.
  2. 모든 입력이 끝나면 저장합니다.
🚨 응시자 목록 전체 노출 제한
개인정보 보호를 위해 응시자 저장 후 해당 페이지에 재진입할 경우, 기존 리스트는 화면에 노출되지 않으며 우측 상단에 '현재 등록 인원: 00명'으로만 간략히 표시됩니다. 등록된 응시자의 개별 수정 및 삭제는 [응시자 관리] 메뉴를 통해서만 가능합니다.
🔒 응시자 접속 화면 및 2차 인증 유의사항

응시자가 검사 사이트에 접속할 때, 사전에 관리자가 설정한 2차 인증 수단으로 본인 확인 절차가 추가됩니다.

  1. 재접속 시 재인증: 응시자가 검사 도중 응시를 중단하고 추후 다시 접속하는 경우, 보안을 위해 선택된 인증 수단으로 다시 2차 인증을 받아야 합니다.
  2. 인증 정보 변경 불가: 응시자 본인이 직접 2차 인증 정보(휴대폰 번호, 이메일 등)를 화면에서 변경할 수 없습니다. 반드시 기업 담당자가 사전에 등록/설정한 정보로만 인증이 가능하므로 최초 등록 전 정확한 정보 입력이 필수입니다.
⚠️ [중요] 2차 인증 5회 오류 시 접근 제한

응시자가 2차 인증 번호 입력을 5회 이상 실패할 경우, 해당 시점으로부터 30분간 검사 사이트 접근이 차단됩니다.
※ 이 차단 조치는 관리자 권한으로도 임의 해제(초기화)가 불가능하므로, 응시자에게 사전에 정확한 본인 정보 입력을 당부해 주시기 바랍니다.

4. 신규 검사 등록

Step 4. 안내 문자/메일 발송

안내 문자/메일 발송 단계에서는 등록한 응시자에게 문자메시지 또는 이메일을 발송할 수 있습니다.

  1. 개인정보 수집동의 안내: 내용을 확인하고 동의합니다. (체크하지 않을 시 발송 제한)
  2. 이메일/문자메시지 선택: 발송할 수단을 선택합니다.
  3. 발송 유형 선택:
    • 즉시: 지금 바로 발송합니다.
    • 예약: 설정한 전송 날짜/시간에 맞춰 발송합니다.
  4. 메일/메시지 발송 내역 확인: 발송 대상 및 내용을 최종 확인합니다.
  5. 즉시/예약 발송 클릭: 안내를 위해 반드시 클릭해야 합니다.
  6. 저장 후 다음 버튼을 누릅니다.

5. 결과 확인 및 응시자 관리

검사가 진행되거나 완료되면 [채용검사 관리] 또는 [응시자 관리] 메뉴에서 실시간 결과 확인 및 성적표 다운로드가 가능합니다.

1. 응시 현황 확인 및 안내 재발송 (홈 화면)

채용검사 관리(홈 화면)에서 전체적인 현황을 확인하고 미응시자에게 안내를 재발송하는 방법입니다.

  • 현황 확인"전체인원" 숫자를 클릭하면 응시자 리스트 팝업(최대 100명 노출)에서 상태별(대기중, 응시중, 검사종료) 인원을 확인할 수 있습니다.
  • 재발송좌측 체크박스를 통해 원하는 인원만 선택하여 문자/메일 발송이 가능합니다. (미응시자 재발송 시 유용)

2. 응시자 정보 보호 및 데이터 파기 관리 (신규)

개인정보 보호법 준수를 위해 응시자 관리 메뉴 내 데이터 파기 관련 기능이 강화되었습니다.

⚠️ 파기 임박 알림 및 데이터 수정/삭제 제한
  • 파기 임박 표시: 이름 및 결과지 파기 일자가 30일 이내로 남은 응시자는 목록에 붉은색 배경으로 강렬하게 표시됩니다. (응시 날짜 클릭 시 파기 예정일 확인 가능)
  • 수정/초기화 불가: 설정된 최소 보관 기간(연락처 보관 기간)이 경과하여 일부 데이터가 이미 파기된 응시자는 개인정보 보호를 위해 '데이터 수정', '응시자 삭제', '응시/시험 초기화' 기능이 모두 비활성화됩니다.

3. 결과 확인 및 다운로드

  • 홈 화면의 "다운로드" 버튼을 클릭하거나 [응시자 관리] 탭에서 PDF/Excel 다운로드를 진행합니다.
  • 결과지 설정: 결과표에 노출하고 싶지 않은 항목을 제외할 수 있습니다.
  • 부적격 판별 기준: 기업 기준에 맞춰 과락 점수를 조정할 수 있습니다. (반드시 응시 전 설정 필요)
  • 주의: PDF 다운로드는 시스템 부하 방지를 위해 최대 50명까지만 가능합니다.

6. FAQ

FIT 관리자 시스템 사용 중 자주 발생하는 문의사항을 정리했습니다.

Q. 아이디/비밀번호를 분실했습니다. 어떻게 찾아야 하나요?
아이디 및 비밀번호 분실 시 시스템 문의처(02-6377-2306, 2308) 또는 담당자 메일(consulting_test@saramin.co.kr)로 문의해 주시기 바랍니다.
Q. 응시자가 2차 인증을 계속 실패하여 접속이 차단되었다고 합니다. 관리자가 해제할 수 있나요?
아니요, 불가능합니다. 보안 정책에 따라 2차 인증 번호 입력을 5회 이상 실패할 경우 30분간 검사 사이트 접근이 차단되며, 이는 관리자 권한으로도 임의 해제나 초기화가 불가합니다. 30분이 경과한 후 사전에 등록된 정확한 본인 정보로 다시 시도하도록 안내해 주시기 바랍니다.
Q. 응시자 관리 목록에서 이름과 연락처가 *** 로 가려져서 보입니다.
개인정보 유출 방지를 위해 '보안 모드(마스킹)'가 기본 적용되어 있기 때문입니다. 마스킹을 해제하고 전체 정보를 확인하시려면, 화면 우측 상단의 [계정 정보] 메뉴에서 [상세 보기 모드]로 전환해 주시기 바랍니다.
Q. 엑셀 일괄 등록 시 자꾸 실패합니다. 오류가 없는 인원만 먼저 등록할 수는 없나요?
현재 시스템은 데이터 누락 방지를 위해 '전체 성공 또는 전체 실패(All-or-Nothing)' 방식을 적용하고 있습니다. 파일 내 단 한 건이라도 필수값(이름, 수험번호, 2차 인증용 연락처/이메일 등)이 누락되거나 형식이 어긋나면 전체 등록이 실패하므로, 오류 내용을 확인하시어 엑셀 데이터를 모두 수정한 후 한 번에 업로드해 주시기 바랍니다.
Q. 응시자 정보를 잘못 입력했습니다. 수정이 가능한가요?
응시자가 아직 검사를 시작하지 않았다면 [응시자 관리] 메뉴에서 개별 응시자의 정보를 변경해 주세요. 단, 이미 파기 기간이 도래하여 일부 데이터가 파기되었거나, 이미 응시 중/완료된 경우에는 수정이 제한될 수 있습니다.
※ 주의: 검사 생성 시 설정한 '2차 인증 수단'과 '데이터 보관 기간'은 응시자가 1명이라도 등록되면 검사 관리 설정에서 변경이 원천 차단됩니다.
Q. 개인정보 보관 기간을 연장할 수 있나요?
응시자가 1명이라도 등록된 이후에는 설정된 데이터 보관 기간을 임의로 수정할 수 없습니다. 설정된 기간 만료 시 데이터가 영구 파기되므로 별도 보관이 필요한 경우 만료 전에 데이터를 미리 백업해 주셔야 합니다.
Q. 결과표(PDF) 다운로드가 안 됩니다.
브라우저의 팝업 차단 기능이 활성화되어 있는지 확인해 주세요. 팝업을 허용하신 후 다시 시도해 보시기 바라며, 시스템 부하 방지를 위해 한 번에 최대 50명까지 다운로드 가능합니다.
Q. 부적격 판별 기준 점수는 언제 설정해야 하나요?
결과표 도출 오류를 방지하기 위해 반드시 응시자가 응시하기 전에 기준을 설정하거나 수정해야 합니다. 부득이하게 응시 후 수정이 필요한 경우 시스템 문의처로 연락 부탁드립니다.
Q. 내부 테스트 진행 시 유의사항이 있나요?
내부 TEST로 진행하시는 경우, 검사명 또는 지원자 이름에 "TEST", "테스트" 등으로 명확히 기재하여 주시기 바랍니다.
Q. 검사 삭제나 강제 채점, 시험 초기화는 어떻게 하나요?
응시 횟수 초과 해제 등 기본적인 관리 기능은 페이지 내에서 가능합니다. 단, 데이터 파기가 이미 진행된 응시자는 시험 초기화 기능을 사용할 수 없습니다. 이 외의 검사 전체 삭제나 강제 채점 등의 기능은 시스템 문의처로 직접 요청해 주시면 처리해 드립니다.
Q. 대금 청구 및 세금계산서 발행 절차가 궁금합니다.
월별 이용 내역을 집계하여 다음 달 첫 영업일에 내역서와 함께 메일로 회신드립니다. 내역 확인 후 이상이 없으면 세금계산서를 발행해 드리며, 지정 계좌로 입금해 주시면 됩니다. (건별 정산 필요시 별도 문의)
Q. 검사 비용 과금 기준은 무엇인가요?
검사가 종료되어 결과지가 생성된 응시자에 한하여 과금됩니다. 등록했으나 미응시한 경우(대기 중)나, 중도 포기로 결과지가 나오지 않은 경우에는 비용이 발생하지 않습니다.
